CSS Cheat Sheet

15+ essential properties and concepts

  • color - Text color
  • background-color - Background color
  • font-size - Size of text
  • font-family - Font style
  • margin - Outer spacing
  • padding - Inner spacing
  • border - Border style and width
  • box-shadow - Adds shadow to elements
  • display - Layout type (block, flex, etc.)
  • flex - Flexbox layout styling
  • position - Absolute, relative, etc.
  • z-index - Stack order of elements
  • overflow - Scroll or hide overflow content
  • text-align - Text alignment
  • hover - Pseudo-class for interaction
  • transition - Smooth change of values
