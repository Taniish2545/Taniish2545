HTML Cheat Sheet

15+ useful tags & properties with examples

  • <html> - Root element of an HTML page
  • <head> - Metadata about the document
  • <title> - Title of the document
  • <meta> - Metadata (charset, viewport, etc.)
  • <link> - Link to external resources (CSS)
  • <script> - JavaScript embedding
  • <body> - Body content
  • <h1> to <h6> - Heading levels
  • <p> - Paragraph
  • <a href> - Anchor/Link
  • <img> - Image tag
  • <ul>, <ol>, <li> - Lists
  • <div> - Generic container
  • <span> - Inline container
  • <table> - Table element
  • <form> - HTML forms
  • <input>, <textarea>, <button> - Form inputs
